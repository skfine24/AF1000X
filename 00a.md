# AF1000X v00a Key Features

## Overview
AF1000X v00a is an educational micro-drone firmware focused on understanding the full system from hardware to flight control.

## Core Features
1. IMU-based attitude stabilization (Roll / Pitch / Yaw)
2. Fused altitude estimation and hovering using ToF + barometer (SPL06)
3. Optical flow (PMW3901) position correction
4. nRF24L01 2.4GHz wireless link, binding, and hop table
5. Battery voltage monitoring with warning/critical handling
6. LED state indication (boot/bind/low-batt/gyro/headless/auto-tune)
7. Serial command support and IMU streaming (`1510` start, `1511` stop)

## Automated Features
1. Auto takeoff / auto landing
2. Hover learning and automatic `hoverThrottle` adjustment and persistence
3. Optical flow auto calibration (FlowK)
4. Sensor POST and auto initialization (IMU/ToF/Baro/Flow)
5. Failsafe on link loss: hold then land
6. Low-battery warning and emergency landing handling
7. Auto tuning after hover ready (altitude PD + yaw P)

# AF1000X v00a 주요 기능

## 개요
AF1000X v00a는 교육용 마이크로 드론 펌웨어로, 하드웨어부터 비행 제어까지 전체 시스템 학습을 목표로 합니다.

## 핵심 기능
1. IMU 기반 자세 안정화 (Roll / Pitch / Yaw)
2. ToF + 기압 센서(SPL06) 융합 고도 추정 및 호버링
3. 옵티컬 플로우(PMW3901) 기반 위치 보정
4. nRF24L01 2.4GHz 무선 링크, 바인딩, 홉 테이블
5. 배터리 전압 모니터링 및 경고/임계 처리
6. LED 상태 표시 (부팅/바인딩/저전압/자이로/헤드리스/오토튠)
7. 시리얼 명령 지원 및 IMU 스트리밍 (`1510` 시작, `1511` 중지)

## 자동화 기능
1. 자동 이륙 / 자동 착륙
2. 호버 학습 및 `hoverThrottle` 자동 보정/저장
3. 옵티컬 플로우 자동 캘리브레이션(FlowK)
4. 센서 POST 및 자동 초기화 (IMU/ToF/Baro/Flow)
5. 링크 손실 페일세이프: 유지 후 착륙
6. 저전압 경고 및 비상 착륙 처리
7. 호버 안정 이후 오토튠 (고도 PD + 요 P)

# AF1000X v00a 主要機能

## 概要
AF1000X v00aは教育用マイクロドローン向けファームウェアで、ハードウェアから飛行制御まで全体の理解を目的としています。

## 主要機能
1. IMUベースの姿勢安定化（Roll / Pitch / Yaw）
2. ToF + 気圧センサ（SPL06）による高度推定とホバリング
3. オプティカルフロー（PMW3901）による位置補正
4. nRF24L01 2.4GHz 無線リンク、バインド、ホップテーブル
5. バッテリー電圧監視と警告/クリティカル処理
6. LEDステータス表示（起動/バインド/低電圧/ジャイロ/ヘッドレス/オートチューン）
7. シリアルコマンド対応とIMUストリーミング（`1510`開始、`1511`停止）

## 自動化機能
1. 自動離陸 / 自動着陸
2. ホバー学習と`hoverThrottle`の自動補正・保存
3. オプティカルフロー自動キャリブレーション（FlowK）
4. センサーPOSTと自動初期化（IMU/ToF/Baro/Flow）
5. リンク喪失フェイルセーフ：維持後に着陸
6. 低電圧警告と緊急着陸処理
7. ホバー安定後のオートチューン（高度PD + ヨーP）
